# eodi.me

> Understand Neighborhoods Before You Arrive

Offline desktop app for neighborhood vibe intelligence — activity, atmosphere, convenience, and more.  
No cloud required. Your data stays on your machine.

---

## Download

Go to [**Releases**](https://github.com/eodi-me/app/releases/latest) to download the latest version.

| Platform | File |
|----------|------|
| Windows | `eodi-me_*_x64-setup.exe` |
| macOS (Apple Silicon + Intel) | `eodi-me_*.dmg` |

---

## Getting Started

1. Download the installer for your platform above
2. Run the installer
3. Launch **eodi.me**
4. Activate with your license key from [eodi.me](https://eodi.me)
